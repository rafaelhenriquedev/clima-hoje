Clima Hoje
Descrição

Clima Hoje é uma aplicação front-end desenvolvida em HTML e CSS que apresenta uma interface moderna e responsiva para visualização de informações meteorológicas. O projeto simula um painel de previsão do tempo com dados de temperatura, previsão por hora, previsão semanal e detalhes climáticos, utilizando um layout limpo, organizado e adaptável a diferentes dispositivos.

O objetivo do projeto é demonstrar habilidades em estruturação de páginas web, organização de layout com Grid e Flexbox, responsividade com Media Queries e padronização visual utilizando boas práticas de desenvolvimento front-end.

Funcionalidades
Interface moderna e minimalista
Layout dividido em duas colunas
Barra de busca de cidades
Previsão do tempo atual
Previsão por hora
Detalhes climáticos (sensação térmica, chuva, vento e índice UV)
Previsão semanal
Design responsivo para dispositivos móveis
Uso de fontes externas (Google Fonts)

Layout da Aplicação

A aplicação é organizada em duas colunas principais:

Detalhes climáticos

Responsividade

O projeto foi desenvolvido com foco em adaptação para diferentes tamanhos de tela.

Desktop
Layout em duas colunas
Espaçamento amplo
Previsão por hora distribuída horizontalmente
Mobile
Layout em coluna única
Elementos centralizados
Previsão por hora com rolagem horizontal
